# How to Run game  

```
pip install -r requirements.txt 
```
*`or`*
```
pip install pygame
```
* `python run.py` *or* `python main.py`

## How to play

| Hoykey | Description |
| :--: | :--: |
| leftkey | moves left |
| rightkey | moves right |
| Space | jump |
| Esc  | Exits game|
